Hypertext Markup Language - HTML


Let's take a look at some of the tags we can use to format our web pages:

  • <p></p> paragraph text
  • <b></b> bold text
  • <em></em> emphasise text
  • <u></u> underline text
  • <ul></ul> creates an unordered (bulleted) list
  • <ol></ol> creates an ordered (numbered) list
  • <li></li> adds an item to the list created

All text should be enclosed between the corresponding opening and closing tags. Tags can also be nested to use multiple formatting options.
